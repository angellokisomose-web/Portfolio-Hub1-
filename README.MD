# Portfolio Download Website

This is a simple static website that shows:

- your profile details
- your work/projects
- Google Drive links for downloading project files

## How to use it

1. Open `script.js`.
2. Replace the placeholder profile details in `siteData.profile`.
3. Replace each `REPLACE_WITH_FILE_ID` link in `siteData.projects` with your real Google Drive file links.
4. Open `index.html` in your browser to view the website.

## Google Drive tip

For downloads to work well, make sure each Google Drive file is shared so visitors can access it.

You can use links like:

`https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing`

If you want a more direct download style link, you can also use:

`https://drive.google.com/uc?export=download&id=YOUR_FILE_ID`
